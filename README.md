# TransitOps - Smart Transport Operations Platform

A backend system for managing fleet vehicles, drivers, trips, maintenance, and operational costs — built for the Odoo Hackathon.

## Tech Stack
- Backend: Python, Flask, SQLAlchemy
- Database: SQLite
- Frontend: HTML,CSS, JavaScript.

## Features
- Vehicle and Driver management
- Trip dispatch with business rule validation (cargo weight, license expiry, availability checks)
- Maintenance tracking with automatic status switching
- Fuel and expense logging with cost calculation
- Dashboard KPIs and fleet utilization reporting
- Vehicle retirement

## How to Run
1. Install dependencies: `pip install flask flask_sqlalchemy flask-cors`
2. Run the server: `python app.py`
3. Server runs at `http://127.0.0.1:5000`
